== Short Description ==
The Plugin is made to easily add Local Schema tag for Local SEO benefits to Post and pages.

== Description ==

This plugin takes input from Site Owner and add Local Schema to pages for better SERP results.

== Installation ==

1. Upload the `Local SEO Schema Markup` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Local SEO schema Menu and setup all the fields.
4. Click the "Save" button and add the Local Schema Tag to post and pages.
